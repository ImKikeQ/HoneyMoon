# Luna de Miel Sonia & Kike

## Estructura
- `index.html`: página principal
- `css/style.css`: estilos responsive
- `/images/`: fotos de portada, regalos y QR

## Cómo abrir
1. Abre `index.html` con doble clic.
2. Prueba los botones de pago: Revolut, Bizum y PayPal.
3. Personaliza imágenes y textos cuando quieras.

## Publicar gratis
Puedes subir este proyecto a [GitHub Pages](https://pages.github.com/) o [Netlify](https://www.netlify.com/).

### Ejemplo de URL sugerida
`honeymoon-soniaykike.netlify.app`
